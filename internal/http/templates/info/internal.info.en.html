Data Browser Matsch | Mazia!

Welcome to the Data Browser Matsch | Mazia! This app provides a user-friendly interface to download meteorological and biophysical variables of the long-term socio-ecological research LT(S)ER site Matschertal / Val di Mazia

To begin

Once selected the measurements and the stations and you have chosen a date range, you are ready to download the data. Available table format are CSV (comma separated values) in a wide ("friendly") or long format. The users of the Scientific Network SouthTyrol can also download R and Python code template to query the database with RStudio or any Phyton IDE.
